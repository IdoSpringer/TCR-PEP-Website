Instructions for running the T cell activation predictor.


  • 1. Specify input
    The input file must be in the same template as the example. (case sensitive)
    The file should be a table with three columns, separated by spaces or tabs.
    The first column is the gene name where the mutation occurred e.g: PAFAH1B1, AKT1
    The second column is the 9-mer in one-letter amino acid codes in capital.
    The third column is the substitution amino acid in the following format: p.L511F (i.e. position 511 was mutated from L to F).

  • 2. Submit to prediction
    Click on the submit button to start the prediction

  • 3. Download results
