Instructions for running the T cell activation predictor.


  • 1. Specify input
    The input file must be in the same template as the example. (case sensitive)
    The file should be a table with two columns, separated by tabs.
    The first column is TCRs CDR3-beta sequences, e.g: CASGDGNQAPLF
    When using the Autoencoder based model, TCR length should be at the most 28. The model will ignore longer sequences.
    The second column is the peptides, e.g: SYIPSAEKI .

  • 2. Submit to prediction
    Click on the submit button to start the prediction

  • 3. Download results
